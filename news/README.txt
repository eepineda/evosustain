EvoSustain Website Package

Contents:
- index.html  (homepage with News integration)
- /news       (professional Journal/News module with automation)

Setup:
1. Upload all files to your hosting.
2. Make sure /news/posts.json exists (generated by build.py).
3. Publish new news:
   - Add Markdown in news/posts/
   - Run: python news/build.py
   - Upload updated files

Menu already includes News link.
Homepage automatically shows latest posts.
